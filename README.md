# Tugas-masa-depan-Ilmu-Komputer-Universitas-Pendidikan-Indonesia-2025
Tugas masa depan Universitas Pendidikan Indonesia 2025 dengan format 3 file berupa header, mesin dan main. Header berupa daftar fungsi dan prosedur yang di gunakan, mesin berupa prosedur dan fungsi, dan main berupa masukan dan pemanggilan prosedur fungsi.
